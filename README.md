# 🎯 StackMatch

**StackMatch** is a full-stack, AI-powered web application that helps job seekers understand exactly how well they fit a job description. By analyzing your existing skills against real job postings, StackMatch reveals your "Match Score," highlights missing skills, provides actionable resume tips, and suggests targeted learning resources.

## ✨ Features

- 🔐 **Secure Authentication**: User sign-up and login utilizing JSON Web Tokens (JWT) and Bcrypt.
- ⚡ **AI Job Analyzer**: Paste any job description to get a comprehensive match analysis.
- 📊 **Visual Insights**: View your skill competency and job gap visually through beautiful interactive Radar Charts.
- 🛠️ **Skill Management**: Dynamically add and manage your current tech stack.
- 🚀 **Personalized Tips**: Get tailored resume tips indicating exactly what you need to emphasize.
- 📚 **Learning Paths**: Receive tool-specific learning suggestions to fill any missing skill gaps.
- 🌗 **Dark Mode Design**: A premium, modern interface built with Framer Motion and Tailwind CSS.

## 🛠️ Tech Stack

**Frontend**
- React 19 (Vite)
- Tailwind CSS
- Recharts (for Data Visualization)
- Framer Motion (for UI Animations)
- Lucide React (for Icons)
- Axios

**Backend**
- Node.js & Express
- MongoDB (Mongoose)
- JSON Web Tokens (JWT)
- Bcrypt (Password Hashing)
- dotenv (Environment Management)

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas Atlas cluster)
